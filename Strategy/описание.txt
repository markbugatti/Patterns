

Есть абстрактный класс персонажей, который реализовуют конкретные классы-наследники. Они просто инициализируют свойство "тип перонажа" из родительского класса.

Стратегия предолагает интерфейс IMove. Оно означает передвижение. Есть разные виды передвижений: летать, идти, ходить и летать. Передвигаться можно по одному и группой.

Для этого Интерфейс IMove будут реализовывать классы Fly, Walk, FlyAndWalk.

Интерфейс предполагает 2 метода передвижения: одиночный Move и для группы MoveGroup
